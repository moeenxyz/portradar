# Contributing to PortRadar

Thanks for contributing.

## Local setup

```bash
swift build
swift run
./scripts/build-app.sh
```

## Guidelines

- keep the app lightweight
- prefer native macOS APIs over extra dependencies
- preserve the menubar-first workflow
- avoid features that require background services unless clearly justified

## Pull Requests

Please include:

- what changed
- why it changed
- how you tested it
- screenshots if the menu UI changed

Small, focused pull requests are preferred.
